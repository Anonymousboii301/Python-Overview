Python Operators Table with Examples

Type Operator Description Example Usage
Arithmetic + Addition 
a = 5 + 3  # 8
 Used for adding numbers.
- Subtraction 
a = 10 - 4  # 6
 Used for subtracting values.
* Multiplication 
a = 6 * 7  # 42
 Used for multiplying values.
/ Division 
a = 10 / 2  # 5.0
 Returns a floating-point result.
% Modulus 
a = 10 % 3  # 1
 Returns remainder after division.
** Exponentiation 
a = 2 ** 3  # 8
 Raises power of a number.
// Floor Division 
a = 10 // 3  # 3
 Returns integer part of division.
Comparison == Equal to 
a = (5 == 5)  # True
 Checks if two values are equal.
!= Not equal to 
a = (5 != 3)  # True
 Checks if values are not equal.
> Greater than 
a = (6 > 3)  # True
 Checks if left value is greater.
< Less than 
a = (2 < 5)  # True
 Checks if left value is smaller.
>= Greater than or equal to 
a = (5 >= 5)  # True
 Checks if left value is greater or equal.
<= Less than or equal to 
a = (3 <= 4)  # True
 Checks if left value is smaller or equal.
Assignment Operators = Assign 
a = 10
 Assigns a value to a variable.
+= Add and assign 
a += 5  # a = a + 5
 Increments variable by a value.
-= Subtract and assign 
a -= 3  # a = a - 3
 Decrements variable by a value.
*= Multiply and assign 
a *= 2  # a = a * 2
 Multiplies variable by a value.
Logical Operators AND Return True if both are True 
a = (True and False)  # False
 Returns True if both are True.
OR Return True if one of them or both are true 
a = (True or False)  # True
 Returns True if one is True.
NOT Return the Reverse boolean value 
a = not(True)   # False
 Reverse the boolean values
Membership in Checks existence 
a = 'x' in 'hello'  # False
 Checks if value exists in sequence.
not in Checks non-existence 
a = 'z' not in 'hello'  # True
 Checks if value does not exist.
