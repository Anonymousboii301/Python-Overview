Set Operations

Let's take set1 = {1, 2, 3, 4} and set2 = {1, 3, 5}

Operation Expression Example
Union set1 | set2 {1, 2, 3, 4, 5}
Intersection set1 & set2 {1, 3}
Set Difference set1 - set2 {2, 4}
Symmetric Difference set1 ^ set2 {2, 4, 5}
Subset
(Is set1 a subset of set2?)		 set1 <= set2 False
#eg. {1, 2, 3} <= {1, 2, 3}: True
Proper Subset
(Is set1 a proper subset of set2?)		 set1 < set2 False
#eg. {1, 2} < {1, 2, 3}: True
Superset
(Is set1 a superset of set2?)		 set1 >= set2 False
#eg. {1, 2, 3} >= {1, 2, 3}: True
Proper Superset
(Is set1 a proper superset of set2?)		 set1 > set2 False
#eg. {1, 2} > {1}: True
