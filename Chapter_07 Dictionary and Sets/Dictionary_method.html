Python Dictionary Methods — Reference

This file lists common dictionary methods available on Python's dict objects with short descriptions and simple examples. No CSS included — just plain HTML.

Method Description Example
clear() Remove all items from the dictionary (makes it empty). 
d = {1: 'a', 2: 'b'}
d.clear()
# d -> {}
copy() Return a shallow copy of the dictionary. 
orig = {'x': 1}
shallow = orig.copy()
# shallow is {'x': 1}
fromkeys(iterable, value=None) Class method that creates a new dictionary with keys from the iterable and all values set to value (default None). 
keys = ['a','b','c']
d = dict.fromkeys(keys, 0)
# d -> {'a':0, 'b':0, 'c':0}
get(key, default=None) Return the value for key if key is in the dictionary, else default (does not raise). 
d = {'name': 'Ayush'}
print(d.get('name'))        # 'Ayush'
print(d.get('age', 0))      # 0
items() Return a view object of (key, value) pairs. Useful for iteration. 
d = {1:'a', 2:'b'}
for k, v in d.items():
    print(k, v)
# prints "1 a" then "2 b"
keys() Return a view object of the dictionary's keys. 
d = {'a':1}
print(list(d.keys()))
# ['a']
values() Return a view object of the dictionary's values. 
d = {'a':1}
print(list(d.values()))
# [1]
pop(key[, default]) Remove the specified key and return the corresponding value. If key is not found and default not provided, raises KeyError. 
d = {'a':1, 'b':2}
val = d.pop('a')   # val = 1; d -> {'b':2}
val2 = d.pop('z', 0)  # val2 = 0 (no error)
popitem() Remove and return a (key, value) pair. In Python 3.7+ it removes the last inserted item. Raises KeyError if the dict is empty. 
d = {'a':1, 'b':2}
item = d.popitem()  # item might be ('b', 2)
# d updated accordingly
setdefault(key[, default]) If key is in the dictionary, return its value. If not, insert key with a value of default and return default. 
d = {'a':1}
val = d.setdefault('b', 0)  # val = 0; d -> {'a':1, 'b':0'}
print(d['b'])
update([other]) Update the dictionary with the key/value pairs from other, overwriting existing keys. other can be another dict or an iterable of key/value pairs or keyword args. 
d = {'a':1}
d.update({'b':2})
# d -> {'a':1, 'b':2}
d.update([('a', 9)])
# d -> {'a':9, 'b':2}
__contains__(key) (use in) Return True if the dictionary has the given key. This is the implementation behind key in d. 
d = {'x':1}
print('x' in d)   # True
print('y' in d)   # False
dict() Constructor: create a dictionary from mappings, iterables of pairs, or keyword args. 
d1 = dict(a=1, b=2)
d2 = dict([('x', 9)])
# d1 -> {'a':1,'b':2}, d2 -> {'x':9}
comprehension (syntax) Not a method but a concise way to build dictionaries: {k: v for ...}. 
squares = {x: x*x for x in range(5)}
# {0:0, 1:1, 2:4, 3:9, 4:16}

Note: Methods like items(), keys(), and values() return view objects — they reflect changes made to the dictionary.
