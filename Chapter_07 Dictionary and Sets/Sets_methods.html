Python Set Methods

Method Description Example
set.add(x) Adds element 'x' to the set. 
my_set = {1, 2}
my_set.add(3)
# my_set is now {1, 2, 3}
set.update(iterable) Adds all the elements of the 'iterable' to set. 
my_set = {1, 2}
my_set.update([3, 4, 5])
# my_set is now {1, 2, 3, 4, 5}
set.remove(x) Removes element 'x' from set. Raises error if element not found. 
my_set = {1, 2, 3}
my_set.remove(2)
# my_set is now {1, 3}
set.discard(x) Removes element 'x' from set. Does not raise any error if element not found. 
my_set = {1, 2, 3}
my_set.discard(4)
# my_set is still {1, 2, 3}
set.pop() Removes and returns a random element from set. Error in case of empty set. 
my_set = {1, 2, 3}
popped_element = my_set.pop()
# popped_element could be 1, 2, or 3
set.clear() Removes all elements from set. 
my_set = {1, 2, 3}
my_set.clear()
# my_set is now set()
set.copy() Returns a shallow copy of set. 
my_set = {1, 2}
new_set = my_set.copy()
# new_set is a new set with {1, 2}
set1.union(set2) Returns a new set which is the union of set1 and set2. 
set1 = {1, 2, 3}
set2 = {3, 4, 5}
new_set = set1.union(set2)
# new_set is {1, 2, 3, 4, 5}
set1.intersection(set2) Returns a new set which is the intersection of set1 and set2. 
set1 = {1, 2, 3}
set2 = {3, 4, 5}
new_set = set1.intersection(set2)
# new_set is {3}
set1.difference(set2) Returns a new set which is equivalent to set1 - set2. 
set1 = {1, 2, 3}
set2 = {3, 4, 5}
new_set = set1.difference(set2)
# new_set is {1, 2}
set1.difference_update(set2) Removes all the elements of set1 which are present in set2. 
set1 = {1, 2, 3}
set2 = {3, 4, 5}
set1.difference_update(set2)
# set1 is now {1, 2}
set1.symmetric_difference(set2) Returns a new set which is equivalent to set1 ^ set2. 
set1 = {1, 2, 3}
set2 = {3, 4, 5}
new_set = set1.symmetric_difference(set2)
# new_set is {1, 2, 4, 5}
set1.symmetric_difference_update(set2) Updates the set keeping the elements found in either of set1 and set2, but not in both set1 and set2. 
set1 = {1, 2, 3}
set2 = {3, 4, 5}
set1.symmetric_difference_update(set2)
# set1 is now {1, 2, 4, 5}
set1.issubset(set2) Returns 'True' if set1 is a subset of set2. 
set1 = {1, 2}
set2 = {1, 2, 3}
is_subset = set1.issubset(set2)
# is_subset is True
set1.issuperset(set2) Returns 'True' if set1 is a superset of set2. 
set1 = {1, 2, 3}
set2 = {1, 2}
is_superset = set1.issuperset(set2)
# is_superset is True
set1.isdisjoint(set2) Returns 'True' if both sets have no common elements. 
set1 = {1, 2}
set2 = {3, 4}
is_disjoint = set1.isdisjoint(set2)
# is_disjoint is True
