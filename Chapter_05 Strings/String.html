Python String Functions & Methods

1. String Functions (Built-in)

FunctionDescriptionWhat Happens If It HappensWhat Happens If It Doesn't
len(str)Returns the length of the string.Returns the number of characters in the string.Returns 0 if the string is empty.
str()Converts an object to a string.Returns a string representation of the object.Returns an error if the object cannot be converted to a string.
repr(obj)Returns a string representation of an object.Provides a string representation suitable for debugging.Returns a default string representation if the object doesn't have a custom one.
ord(char)Returns the Unicode code point of a character.Returns the Unicode code point (integer) of the character.Raises a TypeError if the input is not a single character.
chr(int)Returns the character for a Unicode code point.Returns a string character corresponding to the code point.Raises a ValueError if the code point is out of range.
ascii(obj)Returns a string containing a printable representation of an object, escaping non-ASCII characters.Converts non-ASCII characters to escape sequences.May return an unchanged string if all characters are ASCII.

bytearray(str, encoding)Converts a string to a mutable bytearray.Returns a bytearray object that is mutable.Raises a TypeError if the string cannot be encoded with the given encoding.

2. String Methods

A. Case Conversion

MethodDescriptionWhat Happens If It HappensWhat Happens If It Doesn't
str.upper()Converts all characters to uppercase.Returns the string in uppercase.Returns the same string if it's already in uppercase.
str.lower()Converts all characters to lowercase.Returns the string in lowercase.Returns the same string if it's already in lowercase.
str.capitalize()Capitalizes the first letter of the string.Returns the string with the first character capitalized.Returns the same string if it's already capitalized or has no alphabetic characters.
str.title()Capitalizes the first letter of each word.Returns the string with each word capitalized.Returns the same string if no words are present or all are already capitalized.
str.swapcase()Swaps uppercase letters to lowercase and vice versa.Returns the string with cases swapped.Returns the same string if all characters are already swapped.
str.casefold()Converts to lowercase, more aggressive than lower().Returns the string in lowercase, aggressively.Returns the same string if it's already in lowercase.

B. Formatting

MethodDescriptionWhat Happens If It HappensWhat Happens If It Doesn't
str.center(width, fillchar)Centers the string within the given width.Returns the string centered within the width, padded with the fill character.Returns the string as is if the width is smaller than the string length.
str.ljust(width, fillchar)Left-aligns the string.Returns the string left-aligned within the given width, padded with the fill character.Returns the string as is if the width is smaller than the string length.
str.rjust(width, fillchar)Right-aligns the string.Returns the string right-aligned within the given width, padded with the fill character.Returns the string as is if the width is smaller than the string length.
str.zfill(width)Pads the string on the left with zeros.Returns the string with leading zeros to match the specified width.Returns the string as is if it's longer than the specified width.
str.expandtabs(tabsize)Replaces tabs with spaces.Returns the string with tabs replaced by spaces.Returns the same string if there are no tabs.
str.format(*args, **kwargs)Formats a string using placeholders {}.Returns the formatted string with placeholders replaced by arguments.Raises an error if there are mismatched placeholders and arguments.
str.format_map(mapping)Similar to format(), but works with a dictionary directly.Returns the formatted string using the dictionary.Raises an error if the dictionary is missing a key for a placeholder.
