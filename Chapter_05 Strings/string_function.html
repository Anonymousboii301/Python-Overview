Python String Functions

Function Description Example
len() Returns the number of characters in the string.
If it happens: Returns the length as an integer value.
If it doesn't happen: If the string is empty, it will return 0. 		len("hello") → 5
capitalize() Converts the first character to uppercase, and the rest to lowercase.
If it happens: "hello" becomes "Hello".
If it doesn't happen: If the string starts with a number or special character, no transformation occurs. 		"hello".capitalize() → "Hello"
title() Capitalizes the first letter of each word in the string.
If it happens: "hello world" becomes "Hello World".
If it doesn't happen: If the string has no words (only spaces or non-letter characters), it returns the string unchanged. 		"hello world".title() → "Hello World"
upper() Converts all characters in the string to uppercase.
If it happens: "hello" becomes "HELLO".
If it doesn't happen: If all characters are already uppercase, no change occurs. 		"hello".upper() → "HELLO"
lower() Converts all characters in the string to lowercase.
If it happens: "HELLO" becomes "hello".
If it doesn't happen: If all characters are already lowercase, no change occurs. 		"HELLO".lower() → "hello"
swapcase() Swaps the case of each character (uppercase becomes lowercase, and vice versa).
If it happens: "Hello" becomes "hELLO".
If it doesn't happen: If all characters are in the same case, they simply get swapped. 		"Hello".swapcase() → "hELLO"
strip() Removes any leading (before) or trailing (after) whitespace characters.
If it happens: " hello " becomes "hello".
If it doesn't happen: If there are no extra spaces, the string stays the same. 		" hello ".strip() → "hello"
find() Finds the first occurrence of a substring and returns its index.
If it happens: It returns the index of the substring's first appearance.
If it doesn't happen: If the substring is not found, it returns -1. 		"hello world".find("world") → 6
rfind() Finds the last occurrence of a substring and returns its index.
If it happens: Returns the index of the last occurrence of the substring.
If it doesn't happen: If the substring is not found, it returns -1. 		"hello world".rfind("o") → 7
count() Returns the number of occurrences of a specified substring.
If it happens: It returns an integer count of how many times the substring appears.
If it doesn't happen: If the substring is absent, it returns 0. 		"hello world".count("o") → 2
replace() Replaces occurrences of a substring with another substring.
If it happens: Replaces all occurrences of the specified substring.
If it doesn't happen: If the substring is not found, the original string remains unchanged. 		"hello world".replace("world", "Python") → "hello Python"
split() Splits the string into a list of substrings based on a delimiter.
If it happens: If a delimiter is found, it returns a list of substrings.
If it doesn't happen: If no delimiter is found, the entire string is returned as a list with a single element. 		"a,b,c".split(",") → ["a", "b", "c"]
rsplit() Splits the string into a list of substrings from the right, based on a delimiter.
If it happens: It splits from the right-hand side of the string.
If it doesn't happen: If no delimiter is found, the string is returned as a list with a single element. 		"a,b,c".rsplit(",", 1) → ["a", "b", "c"]
join() Joins elements of an iterable (like a list) into a string, using a separator.
If it happens: Joins list elements with the given separator.
If it doesn't happen: If the iterable is empty, it returns an empty string. 		", ".join(["a", "b", "c"]) → "a, b, c"
startswith() Checks if the string starts with a specified prefix.
If it happens: Returns `True` if the string starts with the prefix.
If it doesn't happen: Returns `False` if the string does not start with the prefix. 		"hello".startswith("he") → True
endswith() Checks if the string ends with a specified suffix.
If it happens: Returns `True` if the string ends with the suffix.
If it doesn't happen: Returns `False` if the string does not end with the suffix. 		"hello".endswith("lo") → True
isdigit() Checks if all characters in the string are digits.
If it happens: Returns `True` if the string is entirely made up of digits.
If it doesn't happen: Returns `False` if the string contains non-digit characters. 		"123".isdigit() → True
isalpha() Checks if all characters in the string are alphabetic.
If it happens: Returns `True` if the string contains only alphabetic characters.
If it doesn't happen: Returns `False` if the string contains digits or special characters. 		"abc".isalpha() → True
isalnum() Checks if all characters in the string are alphanumeric (letters or digits).
If it happens: Returns `True` if the string contains only letters or digits.
If it doesn't happen: Returns `False` if the string contains special characters or spaces. 		"abc123".isalnum() → True
format() Formats the string by inserting values into placeholders.
If it happens: Inserts the specified values into the placeholders in the string.
If it doesn't happen: If no values are provided for the placeholders, it will leave them empty or throw an error. 		"Hello, {}".format("Batman") → "Hello, Batman"
isupper() Checks if all characters in the string are uppercase.
If it happens: Returns `True` if the string is in uppercase.
If it doesn't happen: Returns `False` if the string contains lowercase letters. 		"HELLO".isupper() → True
islower() Checks if all characters in the string are lowercase.
If it happens: Returns `True` if the string is in lowercase.
If it doesn't happen: Returns `False` if the string contains uppercase letters. 		"hello".islower() → True
isspace() Checks if all characters in the string are whitespace.
If it happens: Returns `True` if the string is made entirely of spaces or tabs.
If it doesn't happen: Returns `False` if the string contains non-whitespace characters. 		" ".isspace() → True
issubstring() Checks if the substring exists in the string.
If it happens: Returns `True` if the substring is present.
If it doesn't happen: Returns `False` if the substring is not found. 		"hello world".__contains__("hello") → True
