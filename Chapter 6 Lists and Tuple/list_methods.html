Python List Methods/Functions

Method/Function Description Example Properties
append(item) Adds an item to the end of the list. 
numbers = [1, 2, 3]
numbers.append(4)
print(numbers)  # Output: [1, 2, 3, 4]
 Modifies the list in place. Does not return any value (returns None).
extend(iterable) Extends the list by adding all items from the given iterable (e.g., another list). 
numbers = [1, 2]
numbers.extend([3, 4])
print(numbers)  # Output: [1, 2, 3, 4]
 Modifies the list in place. Does not return any value (returns None).
insert(index, item) Inserts an item at the specified index. 
letters = ['a', 'b', 'd']
letters.insert(2, 'c')
print(letters)  # Output: ['a', 'b', 'c', 'd']
 Modifies the list in place. Does not return any value (returns None).
remove(item) Removes the first occurrence of the specified item. 
fruits = ['apple', 'banana', 'cherry']
fruits.remove('banana')
print(fruits)  # Output: ['apple', 'cherry']
 Modifies the list in place. Raises a ValueError if the item is not found.
pop([index]) Removes and returns the item at the specified index (last item if index is not provided). 
elements = ['a', 'b', 'c']
removed = elements.pop(1)
print(removed)  # Output: 'b'
print(elements)  # Output: ['a', 'c']
 Modifies the list in place. Returns the removed item. If index is out of range, raises an IndexError.
clear() Removes all items from the list, resulting in an empty list. 
data = [1, 2, 3]
data.clear()
print(data)  # Output: []
 Modifies the list in place. Does not return any value (returns None).
index(item, [start, end]) Returns the index of the first occurrence of the specified item. 
fruits = ['apple', 'banana', 'cherry']
index = fruits.index('banana')
print(index)  # Output: 1
 Raises a ValueError if the item is not found.
count(item) Returns the number of occurrences of the specified item in the list. 
nums = [1, 2, 2, 3]
count = nums.count(2)
print(count)  # Output: 2
 Does not modify the list. Returns an integer.
sort(key=None, reverse=False) Sorts the list in ascending order by default. Supports custom sorting via key. 
nums = [3, 1, 2]
nums.sort()
print(nums)  # Output: [1, 2, 3]
 Modifies the list in place. Can be sorted in reverse order if reverse=True.
reverse() Reverses the elements of the list in place. 
nums = [1, 2, 3]
nums.reverse()
print(nums)  # Output: [3, 2, 1]
 Modifies the list in place. Does not return any value (returns None).
copy() Returns a shallow copy of the list. 
original = [1, 2, 3]
copy_list = original.copy()
print(copy_list)  # Output: [1, 2, 3]
 Does not modify the list. Returns a new list.
len(list) Returns the number of items in the list. 
nums = [1, 2, 3]
length = len(nums)
print(length)  # Output: 3
 Does not modify the list. Returns an integer.
max(list) Returns the maximum value in the list (if all items are comparable). 
nums = [1, 2, 3]
maximum = max(nums)
print(maximum)  # Output: 3
 Does not modify the list. Raises a ValueError if the list is empty.
min(list) Returns the minimum value in the list (if all items are comparable). 
nums = [1, 2, 3]
minimum = min(nums)
print(minimum)  # Output: 1
 Does not modify the list. Raises a ValueError if the list is empty.
sum(list) Returns the sum of all elements in the list (if they are numbers). 
nums = [1, 2, 3]
total = sum(nums)
print(total)  # Output: 6
 Does not modify the list. Returns a number (integer or float).
sorted(list) Returns a new sorted list from the elements of any iterable. 
nums = [3, 1, 2]
sorted_list = sorted(nums)
print(sorted_list)  # Output: [1, 2, 3]
 Does not modify the list. Returns a new sorted list.
list(iterable) Creates a list from an iterable (e.g., tuple, set, string). 
tup = (1, 2, 3)
list_from_tuple = list(tup)
print(list_from_tuple)  # Output: [1, 2, 3]
 Returns a new list. Does not modify the original iterable.
any(list) Returns True if any element of the list is True. 
values = [0, 1, 0]
result = any(values)
print(result)  # Output: True
 Does not modify the list. Returns a boolean.
all(list) Returns True if all elements of the list are True. 
values = [1, 1, 1]
result = all(values)
print(result)  # Output: True
 Does not modify the list. Returns a boolean.
enumerate(list) Returns an enumerate object, providing an index and value. 
fruits = ['apple', 'banana', 'cherry']
for index, fruit in enumerate(fruits):
    print(index, fruit)
    # Output: 0 apple
    #         1 banana
    #         2 cherry
 Returns an enumerate object, which can be iterated over.
filter(function, iterable) Filters elements based on a function, returning a filtered list. 
nums = [1, 2, 3, 4, 5]
filtered = filter(lambda x: x % 2 == 0, nums)
print(list(filtered))  # Output: [2, 4]
 Returns an iterator. Can be converted to a list using list().
map(function, iterable) Applies a function to all items and returns a mapped list. 
nums = [1, 2, 3]
doubled = map(lambda x: x * 2, nums)
print(list(doubled))  # Output: [2, 4, 6]
 Returns an iterator. Can be converted to a list using list().
