Python Escape Sequences

Escape Sequence Description Example
\\ Backslash (used to escape other escape sequences) "This is a backslash: \\"
\' Single Quote (used to include a single quote in a string) "It\'s a beautiful day!"
\" Double Quote (used to include a double quote in a string) "He said, \"Hello!\""
\n Newline (starts a new line) "Hello\nWorld" (prints "Hello" on the first line and "World" on the second line)
\t Tab (inserts a tab space) "Hello\tWorld" (outputs: "Hello World")
\r Carriage Return (moves the cursor to the beginning of the line) "Hello\rWorld" (outputs: "Worldo")
\b Backspace (moves cursor one character back) "Hello\bWorld" (outputs: "HellWorld")
\f Form Feed (moves the cursor to the next page or form) "Hello\fWorld" (depends on the environment)
\v Vertical Tab (moves the cursor down vertically) "Hello\vWorld" (depends on the environment)
\a Bell (produces a bell sound in some systems) "Hello\a" (produces a bell sound if supported)
\000 Octal value (represents a character using its octal value) "\101" (outputs "A")
\xhh Hexadecimal value (represents a character using its hexadecimal value) "\x48" (outputs "H")
